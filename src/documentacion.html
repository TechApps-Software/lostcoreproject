Documentacion para desarrolladores

1.Empezando

Preparacion del entorno de trabajo

LostCore es un proyecto muy sencillo pero que le da la base a cualquier programador de crear su propio navegador Este esta hecho con el framework ElectronJs este corre con el motor V8 entonces este puede usar las dev tools de chromium
Requisitos:
Sistema Operativo: Windows, Ubuntu, MacOs RAM: 4GB Disco Duro: 300MB Requerimientos Adicionales: NodeJs 10 Minimo Conocimientos requeridos: HMTL,JS, Git

Obtencion del codigo fuente

El codigo fuente es almacenado y actualizado en GitHub usted necesitara descargarlo desde Git
debe de ejecutar lo siguiente desde su terminal con Git:

git clone https://github.com/TechAppsSv/LostCore.git


Asi usted tendra el codigo fuente de LostCore

Implemetacion en su computador

LostCore al hacer un proyecto con Electron y Node ustede debe instalara las dependencias o modulos necesarios
debe de ejecutar lo siguiente desde su terminal desde la carpeta donde tiene el proyecto:

npm i


O tambien puede ejecutar estos comandos

npm install


npm audit


npm audit fix


Asi usted tendra el codigo fuente de LostCore

2.Escructura de archivos

Archivos JSON

Este solo posee un archivo JSON que es el PACKAGE.JSON este es fundamental para el funcionamiento si se borra no funcionara su proyecto
Usted deberia de ver un archivo JSON similar o igual al siguiente:

{
"name": "LostCore",
"version": "1.0.1",
"description": "",
"main": "src/index.js",
"scripts": {
"start": "electron src/index.js"
},
"keywords": [],
"author": "",
"license": "ISC",
"dependencies": {
"electron-navigation": "^1.5.8",
"update-electron-app": "^1.5.0"
},
"devDependencies": {
"electron-devtools-installer": "^3.1.1"
}
}

Scripts

Para iniciar el proyecto inicial y sin ninguna modificacion usted desde su terminal en la carpeta del proyectod debe de ejecutar:

npm run start


Asi usted vera que se le abre en una ventana el navegador

Archivos JavScript

El proyecto solo incluuye un archivo de este tipo llamado index.js este es el que se encarga de el tama√±o de las ventanas el archivo que cargara el nombre NO BORRAR porque si no se ejecutara
A continuacion se mostrara las partes mas importantes del archivo JS
Resolucion de la ventana inicial

mainWindow = new BrowserWindow({width: 1920, height: 1080, });


Archivo a cargar al inicio

pathname: path.join(__dirname, 'views/charge.html'),




Archivos HTML

El proyecto esta conformado mayoritareamente de archivos HTML los archivos presentes son:
charge.html: pantalla de inicio
index.html: Pantlla de vista principal
update.html: Vista de acerca de
los archivos charge.html, update.html pueden ser eliminados
index.html:Es donde se ubica una barra lateral con a href para poner accesos directos ademas se encuentra la navegacion con electron-navigation que puede ser modificada facilmente

3.Uso y compilacion y modificacion

Modificacion

La modificacion del proyecto usted podra entender facilmente el codigo sabiendo los Requerimientos ya antes mencionados

Uso

Usted puede usar sin ningun problema LostCore como un navegador normal y corriente iniciandolo con NPM RUN START o puede modificarlo a su gusto

Compilacion

Para la compilacion usted necesitara usar alguna dependencia de NodeJs para compilacion de programas hechos en ElectronJs como por ejemplo: electron-package
